## 플로우차트 테스트
### 라면끓이는 방법! 랄랄랄

```mermaid
	flowchart TB
	A{라면이 있는가?} -->|있다| B(냄비를 준비한다)
		B -->C(머그컵 2잔 분량의 물을 붓는다)
		C -->E(스토브를 키고 물을 끓인다)
		E -->F(적당히 끓으면 스프를 넣는다)
		F -->G(물이 완전히 끓으면 면과 파를 넣는다)
		G -->H((쫄깃한 면을 원하는가?))
        	H -->|원한다| H1(면을 넣었다 뺐다를 반복한다)
        	H -->|원치 않는다| H2(그냥둔다)
        	H1 --> I(보기 좋게 그릇에 담는다)
        	H2 --> I
        	I --> J(맛있게 먹는다)
    	A -->|없다| K(라면을 사러간다)
    		K --> L(원하는 라면을 고른다)
    		L --> M(산다)
    		M --> N(집에간다)
    		N --> A
```
